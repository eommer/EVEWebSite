    EasyVideoEdition is a piece of software allowing users to edit videos without complexity. It was created by a group of six ENSIM students as a school year-long project. The main aim of this project is to provide professors with an easy-to-use tool to create and edit videos of their courses.

  • play_for_workUsing EasyVideoEdition

    • 1. Download EasyVideoEdition from here

    • 2. Launch EasyVideoEdition.exe in the root directory

    • 3. Create a new project

    • 4. Add the first video in the Add tab

    • 5. Follow the rest of instructions

  • view_moduleFeatures

    • Easy and quick video edition

    • All common video extensions accepted

    • Multiple quality options

  • warningRequierements

    • Windows 7 or more required

    • Intel i3 or Amd FX recommanded

    • 5 GB disk space

    EasyVideoEdition is a piece of software allowing users (mainly focus on professors) to edit videos whitout complexity. The main aim of this project is to help professors to create and edit videos of theirs curses. This app is a student build in WPF project made at ENSIM.

    • 2. Launch EasyVideoEdition.exe if the root directory

    • 3. Create a new project

    • 4. Add the first video in the Add tab

    • 5. Follow the rest of instructions
